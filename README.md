# Career Compass

Career Compass is a smart student productivity and career guidance platform designed to help students manage their daily academic tasks, stay consistent, and move toward their career goals with structured planning.


## Features

* 👤 User Authentication (Sign up / Login)
* ✅ Daily Task & To-Do Management
* 📅 Calendar-based Reminder System
* 📝 Upload and Manage Notes by Date
* 🎯 Goal Tracking with Positive/Negative Points
* 📊 Productivity Monitoring
* 📱 Clean and User-Friendly Interface

##  How to Run

1. Clone the repository

   ```bash
   git clone https://github.com/iaman45/Career-Compass.git
   ```

2. Open in your IDE

3. Configure MySQL in `application.properties`

4. Run the Spring Boot application

5. Open browser:

   ```
   http://localhost:8080
   ```
